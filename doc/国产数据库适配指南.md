# 国产数据库适配指南

## 1. 概述

本文档描述 DBGuardian 对国产数据库的适配方案和支持计划。

## 2. 支持的国产数据库

| 数据库 | 厂商 | 状态 | JDBC驱动 | 默认端口 |
|-------|------|------|---------|---------|
| 达梦 (DM) | 武汉达梦 | 🔶 计划支持 | `dm-jdbc` | 5236 |
| 人大金仓 (KingBase) | 人大金仓 | 🔶 计划支持 | `kingbase8` | 54321 |
| 华为高斯 (GaussDB) | 华为 | 🔶 计划支持 | `gauss-jdbc` | 5432 |
| OceanBase | 蚂蚁集团 | 🔶 计划支持 | `oceanbase-client` | 2883 |
| TiDB | PingCAP | 🔶 计划支持 | `mysql-connector-java` | 4000 |
| 神通 | 神通国产 | 🔶 计划支持 | `shentong-jdbc` | 2003 |
| 瀚高 (HighGo) | 瀚高软件 | 🔶 计划支持 | `hgdb-jdbc` | 5866 |
| 优炫 (UXSino) | 优炫软件 | 🔶 计划支持 | `uxsino-jdbc` | 3218 |

## 3. 达梦数据库 (DM)

### 3.1 驱动配置

```xml
<dependency>
    <groupId>com.dameng</groupId>
    <artifactId>DmJdbcDriver18</artifactId>
    <version>8.1.2.192</version>
</dependency>
```

### 3.2 连接配置

```yaml
spring:
  datasource:
    dbguardian:
      adapters:
        current-database: dm
      masters:
        - id: dm-master
          url: jdbc:dm://dm-server:5236/mydb
          username: SYSDBA
          password: password
          driver-class-name: dm.jdbc.driver.DmDriver
```

### 3.3 特定SQL

```sql
-- 健康检查
SELECT 1 FROM DUAL

-- 分页 (DM支持LIMIT OFFSET)
SELECT * FROM users LIMIT 10 OFFSET 20

-- 分页 (DM也支持FETCH)
SELECT * FROM users OFFSET 20 ROWS FETCH NEXT 10 ROWS ONLY
```

## 11. 适配检查清单

每个国产数据库适配需要完成以下工作：

- [ ] JDBC驱动依赖配置
- [ ] 驱动类名确认
- [ ] 连接URL格式验证
- [ ] 健康检查SQL测试
- [ ] 分页SQL语法测试
- [ ] 主从复制配置（如果支持）
- [ ] 故障转移SQL测试
- [ ] 只读模式设置测试
- [ ] 单元测试覆盖
- [ ] 集成测试验证
