# DBGuardian 测试方案

## 1. 测试概述

本文档描述 DBGuardian Spring Boot Starter 的测试策略和用例，覆盖读写分离、故障转移、分布式协调等核心功能。

### 1.1 测试环境要求

| 组件 | 版本 | 说明 |
|-----|------|------|
| JDK | 1.8+ | 推荐 JDK 11+ |
| MySQL | 8.0+ | 主库 + 从库（需开启 binlog） |
| Redis | 6.0+ | 分布式协调 |
| Maven | 3.6+ | 构建工具 |

### 1.2 测试基础设施

```
┌─────────────────────────────────────────────────────────────┐
│                     测试环境架构                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌─────────────┐         ┌─────────────┐                  │
│   │   MySQL     │         │   MySQL     │                  │
│   │   Master    │ ──────► │   Slave     │                  │
│   │  (192.168.3.45) │ 复制  │  (192.168.3.46) │                  │

│   ┌─────────────┐                                           │

└─────────────────────────────────────────────────────────────┘
```

## 3. 测试执行

### 3.1 前置条件检查

```bash
# 检查 MySQL 主从复制状态
mysql> SHOW SLAVE STATUS\G
mysql> SHOW MASTER STATUS;

# 检查 Redis 连接
redis-cli -h 192.168.3.45 -p 6379 ping

# 检查主从复制延迟
mysql> SHOW SLAVE STATUS\G
# 查看 Seconds_Behind_Master
```

### 3.2 执行测试

```bash
# 进入测试项目
cd dbguardian-test

# 编译打包（先编译主项目）
cd ..
mvn clean install -DskipTests
cd dbguardian-test

# 运行所有测试
mvn test

# 运行特定测试类
mvn test -Dtest=ReadWriteSplittingTest

# 运行特定测试方法
mvn test -Dtest=ReadWriteSplittingTest#testSelectRouteToSlave
```

### 3.3 验证测试结果

测试通过标准：
- 所有 TC-* 测试用例通过
- 无 ERROR 级别日志
- 读写分离路由正确
- 故障转移自动完成

## 4. 测试报告
